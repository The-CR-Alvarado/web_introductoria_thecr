Transparencia del Software

Lo que hace

Nuestro software prioriza la accesibilidad y la seguridad:

  • Ejecución 100% offline tras la instalación.
  • Adaptación de UX según el perfil de usuario.
  • Feedback auditivo para facilitar la comprensión.

Lo que no hace (aún)

Estamos en constante evolución:

  • No sincroniza datos en la nube (Privacidad total).
  • Reconocimiento por cámara (Versión 2.0).
  • No requiere conexión constante a internet.

Aviso de Privacidad y Seguridad

Protección de Datos

Aisia Learn no recopila, procesa ni almacena información personal fuera del dispositivo móvil. Datos como nombre, edad y rol se almacenan localmente en una base de datos SQLite interna.

Seguridad Local

Al ser una aplicación offline, no existe riesgo de filtraciones externas. Al desinstalar la aplicación, toda la información se elimina permanentemente del sistema de archivos del dispositivo.

Normativa para Menores

Cumplimos con las normativas de protección de menores de Google Play. No solicitamos acceso a contactos, ubicación GPS ni datos sensibles que comprometan la identidad del usuario.

Publicidad y Monetización

La app podrá incluir anuncios filtrados para ser aptos para familias (AdMob). Ofreceremos una opción de pago único para eliminar publicidad, protegiendo siempre la experiencia del niño.

TheCR se compromete a actualizar esta política conforme se agreguen nuevas funciones (como el uso de cámara) para mantener la transparencia total con el usuario.
